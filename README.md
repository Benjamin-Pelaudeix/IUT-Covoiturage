# IUT Covoiturage

## Présentation du projet
### Contexte
Ce projet a été conduit en binôme avec Valentin Rougier lors des Travaux Pratiques de semestre 3 pour le module d'apprentissage à **PHP**. Il a été réalisé sur plusieurs semaines et a conduit à une notes pour le compte de ce module.<br>
Note obtenue : 16 / 20 ✅
### Idée générale
Ce projet est un site de mise en relation d'étudiants de l'IUT du Limousin afin de réaliser un covoiturage.
### Fonctionnalités
Voici un aperçu des fonctionnalités implémentées dans le projet :
- Gestion des étudiants et personnels.
- Recherche des trajets existants.
- Gestion des trajets :
  - Ajout d'un trajet
  - Suppression d'un trajet.
- Gestion des villes :
  - Ajout d'une ville
  - Suppression d'une ville
  - Modification d'une ville.
- Formulaire de connexion sécurisé.
